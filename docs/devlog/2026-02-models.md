models.py

Você nunca criar uma classe do zero por isso temos o models.

```bash
class Categoria(models.Model): # A "mãe" aqui é o Model do Django
    nome = models.CharField(max_length=50)
```

---

`AbstractUser`: Vem com regras prontas (username, first_name). Bom para projetos rápidos.

`AbstractBaseUser`: Controle total. Exige que você crie os campos e ensine o Django a gerenciar a criação (Manager).
