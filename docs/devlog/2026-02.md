### tailwindcss@3.4.1

    npm install -D tailwindcss@3.4.1 postcss autoprefixer
    npx tailwindcss init

em tailwind.config.js ajuste >
```bash
/** @type {import('tailwindcss').Config} */
module.exports = {
content: [
    "./agenda/templates/**/*.html",
    "./static/**/*.js",
],
theme: {
    extend: {},
},
plugins: [],
}
```
No terminal: mkdir -p static/src

Crie o arquivo static/src/input.css e adicione as diretivas:

CSS

    @tailwind base;
    @tailwind components;
    @tailwind utilities;

e rode

    npx tailwindcss -i ./static/src/input.css -o ./static/dist/output.css --watch

---watch para o django-browser-reload atualizar sozinho

ou coloque esse npx em package.json como,

```bash
"scripts": {
    "dev": "tailwindcss -i ./static/src/input.css -o ./static/css/output.css --watch"
},
```
pra simplificar e rode **npm run dev**
